# 🎣 AstrBot 钓鱼插件

</div>

原项目地址：<https://github.com/0xNMLSS/astrbot_plugin_fishing>

[![AstrBot](https://img.shields.io/badge/AstrBot-Plugin-orange.svg)](https://github.com/astrbot/astrbot)

## ✨ 功能特点

- **完整的钓鱼主循环**：注册、签到、状态、钓鱼、自动钓鱼、钓鱼记录、钓鱼帮助
- **扩展钓鱼区域系统**：独立钓鱼成本、区域开放时间、稀有鱼配额、通行证要求、区域限定鱼
- **通行证驻留机制**：支持消耗型 24 小时驻留续票与永久票常驻校验
- **高品质鱼系统**：鱼类分为普通品质和高品质，高品质按更高价值结算
- **背包与安全库存**：鱼塘、水族箱、容量升级、稀有度批量转移
- **装备成长**：鱼竿、饰品、鱼饵、道具、精炼、锁定保护、耐久度与损坏卸下
- **玩家市场**：支持鱼、鱼竿、饰品、道具、大宗商品上架，支持匿名上架
- **期货系统**：动态价格、大宗商品、持仓容量、价格历史、市场分析
- **抽卡与擦弹**：卡池、十连、抽卡记录、擦弹奖池与每日次数限制
- **社交对抗**：排行榜、转账、偷鱼、电鱼、驱灵、税收记录、昵称修改
- **科考系统**：多档时长、许可证门槛、团队目标鱼、贡献分配、成员各自领奖
- **玩家 WebUI 与管理员 WebUI**：玩家端默认 `8888`，管理端默认 `7777`
- **玩家安卓端**：Unity引擎开发的玩家可用安卓端app

## 🔧 安装方法

1. 将本插件放入 `data/plugins/` 目录，或通过 AstrBot 插件安装方式从仓库地址安装。
2. 重启 AstrBot 或重新加载插件。
3. 插件启动后会自动执行数据库迁移。
4. 如需管理员后台，请先配置 `webui.secret_key`。
5. 如需 Linux.do OAuth，请补全 `webui.oauth.linuxdo` 配置。

## 🌐 WebUI

### 玩家 WebUI

- 默认端口：`8888`
- 当前页面：首页总览、图鉴、背包、鱼塘、水族箱、市场、商店、期货、抽卡、酒馆、钓鱼区域
- 登录方式：
  - 游戏内账号密码
  - Linux.do OAuth

### 管理员 WebUI

- 默认端口：`7777`
- 当前管理能力：鱼类模板、鱼竿/鱼饵/饰品模板、抽卡池、玩家与库存、市场、商店、区域、称号、CSV 导入导出
