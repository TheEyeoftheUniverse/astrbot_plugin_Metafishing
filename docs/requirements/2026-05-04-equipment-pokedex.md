# 需求冻结：装备图鉴

为 `astrbot_plugin_fishing` 新增装备图鉴。装备范围包括鱼竿、饰品、鱼饵。装备图鉴用于查看玩家已经获得过的装备，不依赖当前是否仍持有该装备。

## 功能要求

- 新增装备获得聚合表，记录 `user_id`、装备类型、模板 ID、首次获得时间、最近获得时间、累计获得数量。
- 鱼竿、饰品、鱼饵的新增入库路径要写入装备获得聚合表。
- 查看装备图鉴或奖励状态时，要把玩家当前背包中的鱼竿、饰品、鱼饵懒同步进聚合表，兼容上线前已有库存。
- 新增 `装备图鉴` 命令，展示装备总进度和三类装备分组进度。
- 新增 `装备图鉴奖励` 命令，统一领取当前所有可领取装备图鉴奖励。
- 玩家 WebUI 增加装备图鉴页面，并在库存导航中可进入。

## 奖励要求

- 鱼类图鉴和装备图鉴都使用节点奖励。
- 25% 之前的节点奖励使用金币，避免前期直接获得大量钻石。
- 25% 及之后的节点奖励使用钻石。
- 100% 节点是断档奖励，单节点金额为 `66666`。
- 奖励领取记录要能防止重复领取。

## 验收标准

- 新玩家获得鱼竿、饰品或鱼饵后，对应装备图鉴条目解锁。
- 老玩家进入装备图鉴后，现有库存会进入已收集记录。
- 未达到节点不可领取，已领取节点不可重复领取。
- 鱼类图鉴奖励展示和发放金额与新规则一致。
- WebUI 能查看装备图鉴、看到奖励进度，并能领取装备图鉴奖励。
