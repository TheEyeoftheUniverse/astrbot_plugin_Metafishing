# 设计整理：钓鱼装备属性分工

## 背景

钓鱼收益与概率已经从“稀有度堆高星”转向“数量、品质、同星级价值倾向”共同成长。装备与鱼饵的属性分工需要同步收敛，避免所有装备位都提供同一组强收益属性，导致后期 build 只有单一最优解。

## 总体目标

- 鱼竿核心属性是“钓鱼成功率”，数量加成。
- 饰品核心属性是“金币加成”（同星级价值偏差）、品质加成。
- 鱼饵核心属性是“钓鱼 CD 减少”，以及数量加成，品质加成，钓鱼成功率，垃圾重抽概率等小修正。鱼饵效果较为强力，但由于是消耗品所以可以接受。
- 默认钓鱼成功率计划从当前较高基础值调整为 `50%`，后续根据鱼竿提升。


## 属性分工

### 鱼竿

鱼竿的最重要属性应是“钓鱼成功率”。

建议职责：

- 主要提供 `钓鱼成功率加成`。
- 可以少量提供 `品质加成`，体现高阶鱼竿的稳定性。
- 可以少量提供 `稀有度加成`，但不应成为高星收益的主要来源。
- 原则上不作为 `金币价值加成` 的主要来源。
- 原则上不作为 `数量加成` 的主要来源。

设计意图：

- 默认成功率降到 `50%` 后，鱼竿成为提高基础钓鱼体验的核心装备。
- 高级鱼竿提升的是“更稳定钓到鱼”，而不是直接把收益倍率拉满。
- 鱼竿和饰品之间形成职责差异，避免同质化。

### 饰品

饰品应偏向收益修饰与 build 差异化。

建议职责：

- 主要提供 `金币价值加成`，影响同星级内中价值池向高价值池转移。
- 可以提供 `数量加成`，但受星级衰减限制。
- 可以提供 `品质加成`，作为收益成长来源之一。
- 可以提供少量 `稀有度加成`，但不应压过鱼竿或鱼饵的定位。
- 不负责钓鱼成功率的主要提升。

设计意图：

- 饰品决定“钓到以后更值钱、更可能高品质、更可能多一条”。
- 收益成长主要从数量与品质发生，而不是从高星概率暴涨发生。
- 饰品可以承载不同流派，例如价值流、数量流、品质流。

### 鱼饵

鱼饵的主要定位调整为“钓鱼 CD 减少”和临时策略选择。

建议职责：

- 主要提供 `钓鱼 CD 减少`。
- 可以提供临时 `成功率加成`，但不应替代鱼竿的核心定位。
- 可以提供短期 `稀有度加成`，用于一次性或限时策略。
- 可以提供 `垃圾鱼减少`，保留当前“命中垃圾鱼后概率重抽一次”的语义。
- 可以提供特定 `价值加成` 或 `数量加成`，但应作为鱼饵特色而不是常规主属性。

设计意图：

- 鱼饵影响“钓鱼节奏”和“这一次/这一段时间的策略”。
- CD 减少让鱼饵适合做消耗品或限时资源，不和鱼竿、饰品争夺长期收益定位。
- 鱼饵保留玩法选择空间，例如速钓、避垃圾、冲稀有、冲数量。

## 概率与收益规则衔接

### 默认成功率

- 默认钓鱼成功率计划调整为 `50%`。
- 成功率成长主要来自鱼竿。
- 鱼饵可以提供短期成功率补正，但不作为长期主来源。

### 稀有度

- 稀有度加成继续只影响 `1-5 星` 的分布。
- `6+ 星` 不受装备、饰品、鱼饵稀有度加成影响。
- 稀有度加成只做温和转移，避免后期 5 星过多。

### 数量

- 数量加成为加法累计。
- 每次钓鱼最多额外增加 `1` 条。
- 数量加成受星级衰减影响：
  - `1-3 星`: `1.0`
  - `4 星`: `0.7`
  - `5 星`: `0.4`
  - `6+ 星`: `0.0`

### 品质

- 品质加成为加法累计。
- 高品质概率 = 总品质加成 × 星级衰减系数。
- 品质加成受星级衰减影响：
  - `1-3 星`: `1.0`
  - `4 星`: `0.6`
  - `5 星`: `0.3`
  - `6+ 星`: `0.1`

### 同星级金币价值

- 同星级内按 `base_value` 分为低/中/高三档价值池。
- 默认池概率为 `20% / 60% / 20%`。
- 价值加成只把概率从中价值池转移到高价值池。
- 低价值池概率始终保留，避免低价值鱼、任务鱼绝迹。

## 当前实现差距

当前后端已经实现：

- 稀有度温和转移，`6+ 星` 独立。
- 数量加法累计，最多额外 `+1`，`6+ 星` 不增产。
- 品质加法累计，并按星级衰减。
- 同星级价值池抽样，低价值池保留。

后续仍需要实现或调整：

- 默认钓鱼成功率从当前值调整为 `50%`。
- 鱼竿模板需要能表达 `钓鱼成功率加成`，或明确复用现有字段的迁移方案。
- 鱼饵模板需要能表达 `钓鱼 CD 减少`，或明确复用现有字段的迁移方案。
- 管理端是否暴露新字段需要单独评估，不纳入本设计整理的实现范围。

## 非目标

- 本文档不直接修改后端逻辑。
- 本文档不调整现有装备、饰品、鱼饵具体数值。
- 本文档不设计管理端表单。
- 本文档不恢复任何鱼重量相关系统。
