# 弹窗商店商品属性展示需求

## Goal

在现有弹窗式商店内，为商品卡片展示购买后实际获得物品的关键属性，帮助玩家在购买前判断装备、鱼饵和道具用途。

## Scope

- 商店弹窗商品卡片。
- `/api/shops/catalog` 返回的商品奖励数据。
- 玩法商店和总商店共用同一弹窗渲染路径。

## Acceptance Criteria

- 鱼竿奖励显示成功率、稀有、品质、数量加成。
- 饰品奖励显示价值、稀有、品质、数量加成。
- 鱼饵奖励显示成功率、垃圾减少、稀有度、CD 减少。
- 鱼饵 CD 减少用百分比语义表达，例如 `-10.0%`。
- 道具奖励无结构化属性时显示 `effect_description` 用途。
- 购买成功后仍沿用现有弹窗内局部刷新，不关闭弹窗。

## Non-Goals

- 不改商店分流、地图入口、连续购买流程。
- 不新增兜底兼容逻辑或静默降级路径。
- 不调整商品数值或数据库迁移。
