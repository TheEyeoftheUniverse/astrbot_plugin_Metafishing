1. 数据库
   - 新增 `user_equipment_stats` 和 `user_equipment_pokedex_reward_claims`。
   - 更新 `schema_latest.sql`，添加增量迁移。

2. 仓储

4. WebUI
   - 新增装备图鉴路由和模板。
